Important FAQ

what is the Document Object Model (DOM) ?

The Document Object Model (DOM) is a programming interface that represents the structure of a web page as a tree of objects, allowing scripts to interact with and update its content, structure, and style.

How do you select an element from the DOM?

You can select an element using `document.getElementById('id')`, `document.querySelector('.class')`, or `document.querySelectorAll('selector')`.

What is event delegation in the context of the Dom, and why is it useful?

Event delegation is a technique where a single event listener is added to a parent element to handle events for multiple child elements. It's useful for better performance and handling dynamically added elements.

How do you manipulate an element's attributes and style using the DOM?

You can manipulate an element's attributes using `setAttribute()` and its style using the `style` property, e.g., `element.setAttribute('id', 'newId')` or `element.style.color = 'red'`.
